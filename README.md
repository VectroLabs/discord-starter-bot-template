## Installation

**1. Clone the repository.**
```bash
git clone https://github.com/VectroLabs/discord-starter-bot-template.git
```

**2. Install the required dependencies.**
```
npm i discord.js
```

**3. Configure the bot**
in [config](./configuration/index.js) file.

**4. Run the bot.**
```bash
node bot.js
```
